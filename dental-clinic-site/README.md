# Dental Clinic Demo Site

Лендинг стоматологической клиники «ДенталПро» — демо-проект для портфолио фрилансера.

## Что реализовано

- Адаптивный лендинг (HTML + CSS + JS)
- Секции: услуги, врачи, цены, отзывы, контакты, форма записи
- Мобильное меню и плавная прокрутка
- Модальное окно подтверждения записи
- Анимации появления элементов при скролле
- SVG-иконки вместо эмодзи
- Доступность: `aria-label`, `aria-expanded`, `focus-visible`

## Как запустить

```bash
cd dental-clinic-site
python -m http.server 8001
```

Или открыть `index.html` в браузере.

## Деплой

Сайт можно развернуть на GitHub Pages, Netlify, Cloudflare Pages или любом статическом хостинге.
