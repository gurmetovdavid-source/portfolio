# Shared Design System

Общая дизайн-система для демо-проектов во фриланс-портфолио.

## Содержимое

- `tokens.css` — цвета, типографика, отступы, скругления, тени
- `components.css` — кнопки, карточки, формы, бейджи, аккордеон

## Использование

Каждый проект импортирует нужные токены и компоненты напрямую в свой `styles.css`. Дизайн-система остаётся центральным источником согласованных значений.
