# Services

I help service businesses build custom web apps, dashboards, and automation. Every project starts with a clear scope and ends with working code deployed to production.

---

## What I offer

### 1. SaaS platforms & business tools
Custom web applications for clinics, salons, agencies, and service companies.

- Scheduling and appointments
- CRM and client/patient management
- Role-based dashboards and admin panels
- Invoicing, reports, and notifications

### 2. Landing pages & marketing sites
Fast, responsive, SEO-friendly sites built with Next.js.

- Clean UI/UX design
- Animations and scroll effects
- Contact forms and integrations
- Sitemap, schema markup, robots.txt

### 3. Telegram & AI bots
Automation for teams, customers, and internal workflows.

- Telegram bots with admin panels
- Appointment booking bots
- Photo/report collection bots
- AI integrations (LLM, voice, notifications)

### 4. DevOps & deployment
Take your app from local code to live server.

- Docker and Docker Compose setup
- VPS deployment (Ubuntu + Nginx + SSL)
- CI/CD with GitHub Actions
- Domain, DNS, and email configuration

## How to start

1. **Send me a message** — describe your problem, not just the feature list.
2. **Free 20-minute call** — I ask questions and propose a solution.
3. **Written proposal** — scope, milestones, price, and payment schedule.
4. **30–50% upfront payment** — then I start coding.
5. **Weekly demos** — you see progress and give feedback.
6. **Final delivery** — code, docs, deployment, and handoff.
