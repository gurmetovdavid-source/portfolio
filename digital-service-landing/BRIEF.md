# Digital Service Landing Page + Payment Form — Freelance Demo Project

Source Upwork job (approximate): "Create a Landing Page & Payment Integration for Digital Service" (Apr 18 2026).
Client asks: simple, effective, high-converting landing page for a $697 digital service with payment integration.

## Implemented features
- Responsive single-page landing
- Hero section with CTA
- Features grid
- Social proof / results section
- Pricing card
- Checkout form with card formatting (demo only)
- FAQ accordion

## How to run
Open `index.html` in a browser or serve the folder statically.

## Note
Payment form is a UI demo. No real payment processor is connected.
